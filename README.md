# GS Front-End – Plataforma Linking

## Resumo do Projeto
A plataforma **Linking** foi desenvolvida como uma solução para conectar profissionais a oportunidades, oferecendo uma interface simples, intuitiva e responsiva.  
O objetivo do projeto é facilitar o networking profissional, permitindo que usuários construam perfis, naveguem entre áreas de atuação e acessem conteúdos de carreira de maneira rápida e objetiva.

Este projeto foi construído com **HTML, CSS e JavaScript**, focado em boas práticas de front-end, organização de código e experiência do usuário.
